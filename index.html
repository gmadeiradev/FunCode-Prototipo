Projeto Fun Code: Complementar e Integrar a Educação Infantil

Conheça o Fun Code

  • Quem somos

    O Projeto Fun Code tem como base incentivar a promoção do ensino de lógica de programação para crianças entre 5 e 10 anos.

  • Nosso diferencial

    Um dos diferencias da plataforma é o desenvolvimento de habilidades socioemocionais nas crianças, por meio de atividades e jogos matemáticos que impulsionam a criatividade e o pensamento.

  • Nosso Método

    No Brasil, a educação digital ainda não tem o alcance que deveria. Diante dessa realidade a proposta do Fun Code uni três aspectos importantes no Método CPC.

Metodologia:CPC

Missão do Projeto Fun Code

Inovação

A plataforma oferece uma nova forma de ensino de lógica e programação, através de um modelo lúdico, com o uso de jogos, atividades e diversão.

Diversão

O Fun Code, por meio dos seus personagens, jogos e atividades busca desenvolver na criança o ato de brincar associado a tecnologia.

Educação

Em decorrência da defasagem no ensino de lógica de programação para crianças, o Fun Code, tem como missão tornar esse conteúdo mais difundido.
