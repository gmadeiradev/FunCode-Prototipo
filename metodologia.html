Fun Code

Metodologia

A Plataforma FUN CODE tem como premissa a Base Nacional Comum Curricular (BNCC). Criando a metodologia com três elementos denominada de CPC como nosso diferencial, baseada nessas competências do (BNCC): a Cultura Digital, Pensamento científico, crítico e criativo, e a Empatia e cooperação. Sendo assim, o FUN CODE procura auxiliar a criança em seu desenvolvimento do raciocínio lógico, treinar a tomada de decisões e a capacidade de solucionar problemas. O uso desse recurso tecnológico tem como foco no desenvolvimento de múltiplas habilidades.
